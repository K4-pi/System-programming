# File Signature Detector

Simple file type detector written in C.

## Features
- binary file reading
- file type detection using magic numbers
- supports common formats like (ZIP, RAR, TAR, ELF, PNG)

## Possible developments
- add more file signatures
- directory scanning

## Why this project?
This project was created to understand how file formats are identified internally:
magic numbers, binary file I/O, and memory comparison.

## Technologies
- stdio / stdlib
- binary file I/O
- memcmp
